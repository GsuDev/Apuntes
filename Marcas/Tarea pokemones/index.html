¿Cómo funciona?

Construido con tecnología JSON para una experiencia fluida

1

Datos JSON

Toda la información de los Pokémon se almacena en formato JSON estructurado y fácil de usar.

         {
          "Id": "25",
          "Name": "Pikachu",
          "Type 1": "Electric",
          "HP": "35",
          "Attack": "55",
          ...
        }
2

Renderizado Dinámico

Nuestro codigo JavaScript convierte los datos JSON en tarjetas visuales interactivas.

3

Paginación

Las tarjetas se muestran en páginas y no todas de golpe por lo que se puede avanzar y retroceder sin cargar todos los datos de golpe
